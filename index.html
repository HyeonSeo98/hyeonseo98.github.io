PROJECT👨‍💻

GENTLE MONSTER(web)

젠틀몬스터 리뉴얼(팀 프로젝트)

젠틀 몬스터 사이트는 기존 메인페이지에서 상품정보제공,접근성 부족하여 고객이 사이트를 접속하면 상품을 쉽게 접근할 수있도록 제작하였고 서브페이지에서 컬러만 다른상품은 상품아래 색상을 mouseover 할수있는 부분을 만들어 사용성을 좋게 만들기위해 제작하였습니다.

JS의 관련 기술 습득,내장함수 setinterval,parentNode, swiper슬라이드 라이브러리활용 방법을 익혔습니다.

➰주요기능

서브페이지 장바구니담기, 찜하기버튼 count, setinterval을 통한 자동클래스추가, 삭제

➰도메인

https://hyeonseo98.github.io/GentleMonster

➰페이지구성

MainPage / SubPage / Detail Page

➰사용기술

HTML / CSS / JS
